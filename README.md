# FastAPI сервис на основе ТЗ

Разработать микросервис для хранения и управления связками "телефон-адрес". Такой сервис может использоваться для кеширования часто запрашиваемой информации.

Отступления по ТЗ, для телефона я использую `Query`, в ТЗ у нас тело запроса, но я решил унифицировать и по моему мнению query лучше подходит.

В теле запроса используется только домашний адрес.

Redis клиент запускается и закрывается используя события lifespan.

## Быстрый старт

1. Убедитесь, что установлен Docker.
2. Перейдите в корневой каталог проекта.
3. Выполните `docker compose up --build`.

Ознакомьтесь с интерактивной документацией по API по адресу `http://localhost:8010/docs`.
